# DreamCars: Leyendas Sobre Ruedas

Un sitio web elegante y moderno que celebra la historia y los íconos de las marcas de automóviles más legendarias del mundo.

## Descripción General

DreamCars es una experiencia web interactiva que combina diseño moderno con contenido informativo sobre las marcas de automóviles más icónicas. El sitio presenta:

- Información detallada sobre marcas legendarias (Ford, Lamborghini, Ferrari, BMW, Mercedes)
- Galería de imágenes de vehículos emblemáticos
- Sistema de búsqueda de marcas
- Tabla comparativa con especificaciones técnicas
- Formulario de contacto
- Diseño responsivo con efectos visuales atractivos

## Instalación y Configuración

### Requisitos Previos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local (opcional, para desarrollo)

### Instalación
1. Descarga o clona el repositorio
2. Descomprime los archivos si es necesario
3. Asegúrate de que la estructura de archivos mantenga esta organización:

├── style.css
├── script.js

## Uso
### Navegación Principal
- **Inicio**: Te lleva al inicio de la página
- **Galería**: Muestra una colección de imágenes de autos en un overlay
- **Desarrollado por**: Muestra información sobre los creadores
- **Tabla Comparativa**: Presenta datos técnicos comparativos de los vehículos
- **Contacto**: Abre un formulario para ponerse en contacto

### Tecnologías Utilizadas
- HTML5 semántico
- CSS3 con variables personalizadas y efectos modernos
- JavaScript vanilla para interacciones
- Diseño responsivo con Grid y Flexbox
- Efectos de revelado al hacer scroll con Intersection Observer

### Inspiración
Este proyecto fue inspirado por la rica historia de la industria automotriz y el deseo de compartir estas leyendas con entusiastas de los automóviles.
